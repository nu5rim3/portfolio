## Tech Stack

**Frontend:** HTML5 / CSS3 / ReactJS / NextJS

## Quick start

Clone the repo

```bash
  git clone https://github.com/nu5rim3/portfolio.git
```

Install Developer Portfolio with npm

```bash
  cd portfolio
  npm install
```

## Run Locally

To run locally, run the following command

```bash
  npm run dev
```

## Deployment

To create a production build

```bash
  npm run build
```
